什麼是CSS

CSS(Cascading Style Sheets)讓瀏覽器知道該如何呈現你的文件（這裡的文件指的是標籤語言組成的文件例如：HTML, SVG, XML）， 而瀏覽器的功用在於將文件視覺化使它可以呈現在螢幕、投影機或印表機上。

CSS如何作用

瀏覽器會將CSS rule套用在文件上，讓文件的顯示外觀發生變化， 一個CSS rule包含兩個部分：

  1. 選擇器

    告訴瀏覽器，哪些元素會受到這條規則的控制。 選擇器會影響到瀏覽器套用CSS rule的效率， 詳細的內容我們之後會探討。

  2. 影響的屬性與數值

    告訴瀏覽器這條CSS rule要更改的屬性有哪些， 要改成什麼數值。

瀏覽器如何處理CSS

當瀏覽器要顯示一個文件之前，它會將文件內容與style資訊合併成DOM(document object model)，再依照DOM的內容顯示給使用者看。 至於DOM的細節，我們不在此敘述，等到之後有機會在瀏覽器上使用Javascript時再談比較適合。
