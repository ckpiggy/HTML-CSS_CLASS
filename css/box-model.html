Box properties 盒狀屬性

所有html元素都是像下圖一樣一層一層的box，包含：content box, padding box, border box 和margin box。 這些box都有各自對應的CSS屬性控制。我們可以打開瀏覽器的開發者模式觀察這些box的狀況。

  • width與height

    這兩個屬性影響content box的寬和高，content box內可以放置文字或是其他html元素。

  • padding

    這個屬性掌管padding box的大小，padding box代表從content到border的距離。而padding屬性其實是 padding top, padding right, padding bottom, padding left的縮寫。例如：

    .my-padding { padding: 10px; } 代表my-padding類別的padding top, padding right, padding bottom, padding left都設為10px。

    .my-padding { padding: 10px 20px; } 代表my-padding類別的padding top, padding bottom 設為10px，padding right, padding left設為20px。

    .my-padding { padding: 10px 15px 5px 20px; } 依序設定my-padding類別的padding top, right, bottom, left （順時針）的數值。

  • border

    border代表元素的邊框，border的預設寬度是0，我們可以藉由改變border屬性來調整邊框外觀。 border屬性會一次調整元素的四個邊框，如果想針對各個邊框調整可以使用border-top, border-right, border-bottom, border-left。

    .my-border { border: 1px solid blue; } 設定my-border類別的邊框寬度1px，邊框樣式為solid（實心），邊框顏色為藍色。

  • margin

    margin box代表這個元素到其他元素之間的間距，和padding的語法一樣， margin一樣是縮寫，設定順序一樣是由上開始順時針方向。

    margin有一個特別的行為叫作margin collapsing，當兩個元素相接時， 兩個元素的間距並不是margin的加總而是最大的margin。
